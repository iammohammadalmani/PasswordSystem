## Login & Registration System (C# WinForms)

This project is a desktop application built using C# and Windows Forms that implements a basic login and registration system.

The application allows users to:
- Register a new account with username and password validation
- Log in using existing credentials
- Display user-friendly error messages for invalid input
- Toggle password visibility (Show / Hide)
- Navigate between multiple forms (Login, Register, Success screen)

### Features
- Input validation (empty fields, minimum length, no spaces)
- Username uniqueness check
- Password confirmation during registration
- Login authentication using stored credentials
- Clean UI with feedback messages
- Multiple WinForms (Form1, Form2, Form3)

### Technologies Used
- C#
- .NET WinForms
- Visual Studio

### Purpose
This project was created to practice object-oriented programming, form navigation, user input validation, and basic authentication logic in a desktop application.
